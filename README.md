# angular-2-socket.io-mysql-chat
Angular 2 chat

First you need create MySQL DataBase and configure in ./models/db.ts

`
git clone https://github.com/LeGront/angular-2-socket.io-mysql-chat.git u4chat
`

`
cd u4chat
`

`
npm i
`

For start Angular 2 
`
ng serve
`

For start Node

`
node server/server.js
`
